# Math Curve Shooter

2D回合制数学曲线射击游戏 — 类似弹弹堂玩法

## 技术栈
- Unity 2022.3 LTS + C#
- 2D URP 渲染

## 快速开始
1. 阅读 [docs/00_项目总览.md](docs/00_项目总览.md)
2. 各模块独立开发文档在 docs/ 目录下

## 版本
- V1: 基础版（场景+玩家+回合射击）
- V2: 单机人机
- V3: 本地多人
- V4: 联机多人
