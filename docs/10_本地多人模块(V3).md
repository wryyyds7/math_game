## 一、需求概述

在V1基础上增加本地多人（热座 Hot-Seat）模式：
1. 2-4名玩家同屏轮流操作
2. 游戏设置画面：选择玩家数量、玩家名称
3. 同一台设备，屏幕保密困难 → 改为共享屏幕，轮流输入
4. 每个玩家操作时，其他玩家可观看（或简单遮挡）

## 三、类设计

### LocalMultiplayerManager (主类)
```csharp
// 文件: Assets/_Project/Scripts/LocalMultiplayer/LocalMultiplayerManager.cs
using System.Collections.Generic;
using UnityEngine;

namespace MathGame.LocalMultiplayer
{
    /// <summary>
    /// 本地多人管理器 — 管理热座模式流程
    /// </summary>
    public class LocalMultiplayerManager : MonoBehaviour, ILocalMultiplayerManager
    {
        [SerializeField] private PlayerSetupUI setupUI;
        [SerializeField] private ITurnManager turnManager;
        [SerializeField] private IPlayerManager playerManager;
        [SerializeField] private IUIManager uiManager;

        private int playerCount = 2;
        private string[] playerNames;

        private void Start()
        {
            SetupPlayers(2, new[] { "玩家1", "玩家2" }); // 默认2人
        }

        public void SetupPlayers(int count, string[] names)
        {
            playerCount = Mathf.Clamp(count, 2, 4);
            playerNames = new string[playerCount];
            for (int i = 0; i < playerCount; i++)
            {
                playerNames[i] = i < names.Length ? names[i] : $"玩家{i + 1}";
            }
        }

        public string[] GetPlayerNames() => playerNames;

        public void StartLocalGame()
        {
            turnManager.StartGame(playerCount);
            uiManager.ShowGameHUD();
        }
    }
}
```

### PlayerSetupUI (玩家设置UI)
```csharp
// 文件: Assets/_Project/Scripts/LocalMultiplayer/PlayerSetupUI.cs
namespace MathGame.LocalMultiplayer
{
    /// <summary>
    /// 游戏前设置画面：选择玩家数量、输入名字、选择颜色
    /// </summary>
    public class PlayerSetupUI : MonoBehaviour
    {
        [Header("玩家数量选择")]
        [SerializeField] private Button[] playerCountButtons; // 2人, 3人, 4人

        [Header("玩家配置")]
        [SerializeField] private GameObject[] playerConfigRows; // 最多4行
        // 每行包含：TMP_InputField (名字), Dropdown (颜色)

        [Header("按钮")]
        [SerializeField] private Button startButton;
        [SerializeField] private Button backButton;

        public event Action<int, string[], Color[]> OnGameStart;
        public event Action OnBack;

        private int selectedCount = 2;

        private void Awake()
        {
            // 绑定按钮事件
            for (int i = 0; i < playerCountButtons.Length; i++)
            {
                int count = i + 2;
                playerCountButtons[i].onClick.AddListener(() => SelectPlayerCount(count));
            }
            startButton.onClick.AddListener(OnStartClicked);
            backButton.onClick.AddListener(() => OnBack?.Invoke());

            SelectPlayerCount(2); // 默认2人
        }

        private void SelectPlayerCount(int count)
        {
            selectedCount = count;
            // 显示对应数量的玩家配置行
            for (int i = 0; i < playerConfigRows.Length; i++)
            {
                playerConfigRows[i].SetActive(i < count);
            }
            // 高亮选中的按钮
        }

        private void OnStartClicked()
        {
            string[] names = new string[selectedCount];
            Color[] colors = new Color[selectedCount];
            for (int i = 0; i < selectedCount; i++)
            {
                var input = playerConfigRows[i].GetComponentInChildren<TMP_InputField>();
                names[i] = string.IsNullOrEmpty(input.text) ? $"玩家{i + 1}" : input.text;
                colors[i] = Color.white; // TODO: 颜色选择
            }
            OnGameStart?.Invoke(selectedCount, names, colors);
        }
    }
}
```

## 四、热座流程设计

```
1. 主菜单 → "本地多人" → 设置画面
2. 选择玩家数量(2-4)，输入名字
3. 点击"开始游戏"
4. 游戏循环：
   a. 显示"轮到 {玩家N}"的提示（可选短暂遮挡）
   b. 显示操作面板给当前玩家
   c. 玩家完成操作后移交设备
   d. 重复直到游戏结束
5. 游戏结束 → 显示排名
```

### 热度安全：状态覆盖提示
- 切换玩家时，显示一个简短的全屏提示 "轮到 玩家X！请交接设备"
- 上一个玩家的输入面板自动隐藏
- 可选：在切换瞬间短暂黑屏（0.5秒），期间上一个玩家可移交设备
