# 00 - 项目总览

## 项目名称
**Math Curve Shooter** — 2D回合制数学曲线射击游戏

## 技术栈
- **引擎**: Unity 2022.3 LTS
- **语言**: C# (.NET Standard 2.1)
- **渲染**: Unity 2D (SpriteRenderer + URP)
- **网络(V4)**: Unity Netcode for GameObjects / Mirror
- **版本管理**: Git + GitHub
- **仓库地址**: https://github.com/wryyyds7/math_game

## 游戏核心玩法

### 场景
- 2D俯视视角场景，固定大小地图（建议1920×1080逻辑像素）
- 场景中随机分布**障碍物**（圆形/矩形，大小随机，不可移动）
- 场景中随机分布**升级星星**（不同大小对应不同经验值）
- 玩家随机出生点，随机初始朝向

### 回合制规则
1. 每回合每个玩家**三选一**行动：
   - **射击**: 输入数学函数，以玩家朝向为 x 轴正方向绘制曲线，子弹沿曲线飞行
   - **闪现**: 瞬移到以玩家为中心、地图范围30%半径内的任意位置（不受障碍物阻挡）
   - **旋转180度**: 改变朝向
2. 每局**2次闪现**机会
3. 子弹碰撞障碍物 → 产生圆形范围伤害（伤害半径随等级增长）
4. 子弹击中玩家 → 该玩家出局
5. 最后存活的玩家获胜

### 等级系统
- 吃星星获得经验值，星星越大经验越多
- 升级经验逐级上涨，公式: `所需经验 = baseExp * level^1.5`
- 等级影响：伤害半径、闪现范围（固定地图30%不变，但等级解锁/增强效果可考虑）

## 版本规划

| 版本 | 内容 | 预计工期 | 独立Agent数 |
|------|------|----------|-------------|
| V1.0 | 基础版：场景生成、单玩家测试、回合制射击机制 | 2周 | 6个并行 |
| V2.0 | 单机人机：AI对手、难度区分 | 1周 | +1 AI模块 |

## 开发规范
1. 所有模块通过接口通信，避免直接依赖
2. 使用 `ScriptableObject` 存储全局配置
3. 事件驱动架构（UnityEvent / C# event）
4. 每个模块独立 Assembly Definition
5. 提交信息格式: `YYYY年MM月DD日HH时，进行了XXX`
