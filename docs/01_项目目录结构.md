## Assembly Definition 规划

每个模块使用独立的 Assembly Definition (`.asmdef`)，确保编译隔离：

| 模块 | Assembly Name | 依赖 |
|------|--------------|------|
| Core.Data | `Game.Core.Data` | 无 |
| Core.Interfaces | `Game.Core.Interfaces` | Game.Core.Data |
| SceneManagement | `Game.SceneManagement` | Game.Core.Interfaces |
| Player | `Game.Player` | Game.Core.Interfaces |
| MathParser | `Game.MathParser` | Game.Core.Interfaces |
| Shooting | `Game.Shooting` | Game.Core.Interfaces, Game.MathParser |
| TurnManagement | `Game.TurnManagement` | Game.Core.Interfaces, Game.Player, Game.Shooting |
| UI | `Game.UI` | Game.Core.Interfaces |
| AI (V2) | `Game.AI` | Game.Core.Interfaces, Game.MathParser |
| LocalMultiplayer (V3) | `Game.LocalMultiplayer` | Game.Core.Interfaces, Game.TurnManagement |
| Network (V4) | `Game.Network` | Game.Core.Interfaces, Game.TurnManagement |

## 命名空间规范

```csharp
namespace MathGame.Core.Data      // 数据结构
namespace MathGame.Core.Interfaces // 接口
namespace MathGame.SceneManagement // 场景
namespace MathGame.Player          // 玩家
namespace MathGame.MathParser      // 数学解析
namespace MathGame.Shooting        // 射击
namespace MathGame.TurnManagement  // 回合
namespace MathGame.UI              // UI
namespace MathGame.AI              // AI (V2)
namespace MathGame.LocalMultiplayer // 本地多人 (V3)
namespace MathGame.Network         // 网络 (V4)
```

## 并行开发建议
- **V1阶段**: Agent-1~6 可完全并行开发各自的模块目录
- **共享代码**: `Core/Data/` 和 `Core/Interfaces/` 需最先完成（由Agent-A负责）
- **集成顺序**: Core → Scene + Player → MathParser + Shooting → TurnManagement + UI
