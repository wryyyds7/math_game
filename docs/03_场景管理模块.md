# 03 - 场景管理模块 (Agent-1)

> **依赖**: 02_核心数据结构
> **被依赖**: 04_玩家模块, 06_射击系统模块, 07_回合管理模块
> **V1交付**: 场景生成器 + 障碍物 + 星星

---

## 一、需求概述

负责2D游戏场景的生成与管理，包含：
1. 固定大小地图边界
2. 随机生成的障碍物（圆形/矩形），保证不重叠且不与玩家出生点冲突
3. 随机生成的升级星星（3种大小），保证不在障碍物内部

## 二、接口定义

```csharp
// 文件: Assets/_Project/Scripts/Core/Interfaces/ISceneManager.cs
using System.Collections.Generic;
using UnityEngine;
using MathGame.Core.Data;

namespace MathGame.Core.Interfaces
{
    public interface ISceneManager
    {
        /// <summary>地图宽度</summary>
        float MapWidth { get; }

        /// <summary>地图高度</summary>
        float MapHeight { get; }

        /// <summary>所有障碍物列表</summary>
        List<ObstacleData> Obstacles { get; }

        /// <summary>所有星星列表</summary>
        List<StarData> Stars { get; }

        /// <summary>生成新地图（随机障碍物+星星）</summary>
        void GenerateMap();

        /// <summary>获取合法出生点（不与其他物体重叠）</summary>
        Vector2 GetValidSpawnPoint();

        /// <summary>检查某位置是否在地图内</summary>
        bool IsInsideMap(Vector2 position);

        /// <summary>检查某位置是否与障碍物碰撞</summary>
        bool IsCollidingWithObstacle(Vector2 position, float radius);

        /// <summary>收集星星（由玩家模块调用）</summary>
        StarData CollectStar(int starID);

        /// <summary>对障碍物造成伤害（由射击系统调用）</summary>
        void DamageObstacle(int obstacleID, float damage);
    }
}
```

## 四、生成算法

### 障碍物生成伪代码
```
1. 初始化空列表 obstacles
2. FOR i = 0 TO count:
3.   DO:
4.     pos = 随机位置(留出地图边距100px)
5.     radius = 随机半径(config.ObstacleMinRadius ~ config.ObstacleMaxRadius)
6.     shape = 随机形状(70%圆, 30%矩形)
7.   WHILE (与已有障碍物重叠)
8.   添加到列表
9. RETURN obstacles
```

### 重叠检测
```
两个圆形：圆心距 > r1 + r2 + 最小间距(20px)
圆与矩形：圆心到矩形最近点距 > r + 外接圆半径 + 最小间距
两个矩形：外接圆检测（简化）
```

### 星星生成
```
1. FOR i = 0 TO count:
2.   DO:
3.     pos = 随机位置(地图内)
4.     size = 加权随机尺寸(60%小,30%中,10%大)
5.   WHILE (pos在任何障碍物内部)
6.   添加到列表
```
