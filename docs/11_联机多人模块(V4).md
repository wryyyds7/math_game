# 11 - 联机多人模块 V4 (Agent-9)

> **依赖**: 02_核心数据结构, 04_玩家模块, 07_回合管理模块, 08_UI模块

## 一、需求概述

在V1基础上实现网络联机对战：
1. 创建/加入房间（Lobby系统）
2. 2-4名玩家联网对战
3. 回合制同步：客户端-服务器权威模式
4. 游戏状态网络同步（玩家位置、子弹、地图）
5. 断线重连处理

---

## 二、技术选型

| 方案 | 优点 | 缺点 |
|------|------|------|
| **Unity Netcode for GameObjects (NGO)** | 官方支持、易上手、托管 | 需要Relay服务（免费额度有限） |
| Mirror | 成熟稳定、社区大 | 非官方 |
| Photon Fusion/PUN | 功能强、云服务 | 收费、学习成本 |

**推荐方案**: **Unity Netcode for GameObjects**（Unity 2022.3 原生支持）

## 四、网络架构

### 权威模型
```
┌──────────────┐     ┌──────────────┐
│   主机/服务器  │◄───►│   客户端1     │
│  (权威逻辑)    │     │              │
│              │◄───►│   客户端2     │
│  地图生成     │     └──────────────┘
│  回合管理     │
│  碰撞检测     │
│  伤害计算     │
└──────────────┘
```

### 同步策略
| 数据 | 同步方式 | 频率 | 权威方 |
|------|---------|------|--------|
| 玩家位置 | NetworkTransform | 实时 | 服务器 |
| 玩家状态（等级、闪现等） | NetworkVariable | 变化时 | 服务器 |
| 子弹 | 自定义NetworkBehaviour | 实时 | 服务器 |
| 曲线数据 | RPC (Reliable) | 一次性 | 客户端→服务器 |
| 回合状态 | NetworkVariable | 变化时 | 服务器 |
| 障碍物 | 仅主机生成→同步 | 一次性 | 主机 |
| 星星 | 仅主机生成→同步 | 一次性 | 主机 |

## 五、类设计

### NetworkGameManager (主网络管理器)
```csharp
// 文件: Assets/_Project/Scripts/Network/NetworkManager.cs
using Unity.Netcode;
using System.Collections.Generic;

namespace MathGame.Network
{
    public class NetworkGameManager : NetworkBehaviour, INetworkManager
    {
        [SerializeField] private ITurnManager turnManager;
        [SerializeField] private IPlayerManager playerManager;
        [SerializeField] private ISceneManager sceneManager;

        // === 网络变量 ===
        private NetworkVariable<GamePhase> networkPhase = new(GamePhase.WaitingForPlayers);
        private NetworkVariable<int> currentTurn = new(0);
        private NetworkVariable<ulong> currentPlayerClientID = new(0);
        private NetworkList<NetworkPlayerInfo> roomPlayers;

        public bool IsConnected => NetworkManager.Singleton.IsConnectedClient;
        public bool IsHost => NetworkManager.Singleton.IsHost;

        private void Awake()
        {
            roomPlayers = new NetworkList<NetworkPlayerInfo>();
        }

        public override void OnNetworkSpawn()
        {
            if (IsHost)
            {
                NetworkManager.Singleton.OnClientConnectedCallback += OnClientConnected;
                NetworkManager.Singleton.OnClientDisconnectCallback += OnClientDisconnected;
            }

            networkPhase.OnValueChanged += OnPhaseChanged;
            currentPlayerClientID.OnValueChanged += OnCurrentPlayerChanged;
        }

        public void CreateRoom(string roomName, int maxPlayers)
        {
            // 使用Unity Relay创建房间
            // 或使用局域网直连
            NetworkManager.Singleton.StartHost();
        }

        public void JoinRoom(string roomCode)
        {
            // 通过Relay加入
            NetworkManager.Singleton.StartClient();
        }

        public void LeaveRoom()
        {
            if (IsHost)
                NetworkManager.Singleton.Shutdown();
            else
                NetworkManager.Singleton.Shutdown();
        }

        public List<NetworkPlayerInfo> GetRoomPlayers()
        {
            var list = new List<NetworkPlayerInfo>();
            foreach (var p in roomPlayers) list.Add(p);
            return list;
        }

        public void StartNetworkGame()
        {
            if (!IsHost) return;
            StartGameServerRpc();
        }

        [ServerRpc(RequireOwnership = false)]
        private void StartGameServerRpc()
        {
            // 服务器权威：启动游戏流程
            sceneManager.GenerateMap();
            
            // 为每个连接客户端生成玩家
            foreach (var client in NetworkManager.Singleton.ConnectedClients)
            {
                var playerState = playerManager.SpawnPlayer(
                    roomPlayers[(int)client.Key].PlayerName);
                // 通过RPC将玩家状态发回客户端
                SpawnPlayerClientRpc(playerState, new ClientRpcParams
                {
                    Send = new ClientRpcSendParams { TargetClientIds = new[] { client.Key } }
                });
            }

            turnManager.StartGame(roomPlayers.Count);
        }

        [ClientRpc]
        private void SpawnPlayerClientRpc(PlayerState state, ClientRpcParams param)
        {
            // 客户端接收玩家生成信息
        }

        /// <summary>客户端提交动作</summary>
        public void SubmitNetworkAction(TurnActionType action, string actionDataJson)
        {
            SubmitActionServerRpc(action, actionDataJson);
        }

        [ServerRpc(RequireOwnership = false)]
        private void SubmitActionServerRpc(TurnActionType action, string actionDataJson,
                                           ServerRpcParams param = default)
        {
            // 服务器验证并执行动作
            int playerID = GetPlayerIDByClient(param.Receive.SenderClientId);
            turnManager.SubmitAction(playerID, action, DeserializeActionData(action, actionDataJson));
        }

        private void OnClientConnected(ulong clientID)
        {
            roomPlayers.Add(new NetworkPlayerInfo
            {
                ClientID = clientID,
                PlayerName = $"玩家{roomPlayers.Count}",
                IsReady = false
            });
        }

        private void OnClientDisconnected(ulong clientID)
        {
            for (int i = roomPlayers.Count - 1; i >= 0; i--)
            {
                if (roomPlayers[i].ClientID == clientID)
                    roomPlayers.RemoveAt(i);
            }
        }

        private void OnPhaseChanged(GamePhase oldPhase, GamePhase newPhase)
        {
            // 所有客户端同步阶段变化
        }

        private void OnCurrentPlayerChanged(ulong old, ulong current)
        {
            // UI更新：高亮当前行动玩家
        }

        private int GetPlayerIDByClient(ulong clientID) { /* 映射 */ return 0; }
        private object DeserializeActionData(TurnActionType action, string json) { return null; }
    }
}
```

### NetworkPlayer (网络玩家实体)
```csharp
// 文件: Assets/_Project/Scripts/Network/NetworkPlayer.cs
using Unity.Netcode;

namespace MathGame.Network
{
    /// <summary>
    /// 网络同步的玩家实体
    /// </summary>
    public class NetworkPlayer : NetworkBehaviour
    {
        public NetworkVariable<Vector2> Position = new();
        public NetworkVariable<float> Rotation = new();
        public NetworkVariable<int> Level = new(1);
        public NetworkVariable<int> BlinkCharges = new(2);
        public NetworkVariable<bool> IsAlive = new(true);

        private PlayerController controller;

        public override void OnNetworkSpawn()
        {
            controller = GetComponent<PlayerController>();

            Position.OnValueChanged += (old, val) => controller?.SetPosition(val);
            Rotation.OnValueChanged += (old, val) => controller?.SetRotation(val);
        }

        [ServerRpc]
        public void RequestBlinkServerRpc(Vector2 target)
        {
            // 服务器验证并执行闪现
        }

        [ServerRpc]
        public void RequestRotateServerRpc()
        {
            Rotation.Value = (Rotation.Value + 180f) % 360f;
        }
    }
}
```

## 六、房间系统流程

```

        主机                                   客户端
        
    创建房间                                    
    ├─ StartHost()                            
    ├─ 生成RoomCode                           
    └─ 等待玩家加入 ────────────────►  输入RoomCode
                                      ├─ StartClient()
      玩家已连接 ◄─────────────────────┘
    │
    玩家就绪 ← 所有人都Ready
    │
    StartNetworkGame()
    ├─ 生成地图(仅主机)
    ├─ 生成所有玩家
    ├─ 同步到客户端
    └─ 开始回合循环
```

## 七、断线重连

```
检测断线 → 暂停游戏
  ├─ 等待重连（30秒倒计时）
  ├─ 重连成功 → 同步完整状态 → 恢复游戏
  └─ 超时 → 该玩家判定淘汰 → 继续游戏
```
